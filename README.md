# Claude Code Skills

A collection of Claude Code skills for PMs who build with AI.

## Available Skills

| Skill | What it does |
|-------|-------------|
| **netlify-deploy** | Deploy static HTML prototypes to Netlify and get shareable URLs. Designed for PMs and non-engineers who need to quickly share mockups with stakeholders. |

## Install

Add this marketplace to Claude Code:

```
/plugin marketplace add derektsoi/claude-skills
```

Then install a skill:

```
/plugin install netlify-deploy@derektsoi-skills
```

## About

These skills come from real PM workflows — built because I needed them, shared because others might too. Read about how I use them on [Substack](https://derektsoi.substack.com).
